# calculator
# 🧮 PZ Calculator

A modern desktop calculator built with **Python**, **Tkinter**, and **CustomTkinter**.

Designed with a clean dark theme, live expression preview, scientific functions, keyboard shortcuts, and calculation history, this project demonstrates GUI development, event handling, and application state management in Python.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/pz-calculator.git
```

Go into the project directory

```bash
cd pz-calculator
```

Install dependencies

```bash
pip install customtkinter
```

Run the project

```bash
python calculator.py
```
